[ プレイヤー ] ※キーボードのみ対応
・移動…W/A/S/Dキー
・ルクミン飛ばす…Lキー
・ルクミン集合の笛…ENTERキー

//==================================================================
// 進捗(実装内容)
//==================================================================
[2024年1月19日 プロトタイプ版]
・ゲーム画面のみ
・プレイヤーの移動
・ルクミン投げる処理(現在はLキーで出現させるだけ)

[2024年1月26日 α版]
・手持ちのルクミンをプレイヤーが向いてる方向に投げる処理(Lキーで投げる)
・投げられるときのルクミンの位置修正
・ルクミンがプレイヤーに追尾する処理
・ルクミンが戻ってくる処理

[2024年2月2日 β版]
・ポイントのプレイヤー追尾処理
・平面上の笛の範囲生成処理
・ルクミン集合処理
・敵がプレイヤーに追尾する処理
・敵とプレイヤーの当たり判定
・敵とプレイヤーが当たった時にリザルトに遷移
・画面遷移
